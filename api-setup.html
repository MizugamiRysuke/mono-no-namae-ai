🤖 Google Vision API セットアップガイド

✨ 高精度AI認識を有効にするためのガイドです
このガイドに従って設定すると、同じ物体は一貫して同じ名前で認識されるようになります。

📋 必要なもの

  • Googleアカウント（Gmail等）
  • Google Cloud Platform（GCP）アカウント
  • クレジットカード（少額の課金設定用）
💰 料金について
Google Vision APIは月1000回まで無料です。一般的な個人利用では無料枠内で十分使用できます。
  • 月1000回まで: 無料
  • 1001回目以降: 1000回あたり$1.50

🚀 セットアップ手順

Step 1: Google Cloud Console にアクセス
  1. Google Cloud Console を開く
  2. Googleアカウントでログイン
  3. 初回の場合、利用規約に同意して無料トライアルを開始
Step 2: 新しいプロジェクトを作成
  1. 画面上部の「プロジェクトを選択」をクリック
  2. 「新しいプロジェクト」をクリック
  3. プロジェクト名を入力（例：「mono-no-namae-ai」）
  4. 「作成」をクリック
Step 3: Vision API を有効化
  1. 左側メニューから「APIとサービス」→「ライブラリ」
  2. 検索ボックスで「Vision」を検索
  3. 「Cloud Vision API」を選択
  4. 「有効にする」ボタンをクリック
Step 4: APIキーを作成
  1. 左側メニューから「APIとサービス」→「認証情報」
  2. 「+ 認証情報を作成」→「APIキー」をクリック
  3. APIキーが生成されます
  4. 「キーを制限」をクリック（セキュリティのため）
  5. 「API の制限」で「Cloud Vision API」のみを選択
  6. 「保存」をクリック
⚠️ セキュリティに関する重要な注意
APIキーは他人に知られないよう大切に管理してください。万が一漏洩した場合は、すぐに無効化して新しいキーを作成してください。
Step 5: アプリでAPIキーを設定
  1. 作成されたAPIキーをコピー
  2. カメラページに戻る
  3. 撮影を試すと「Google Vision API使用しますか？」の確認が表示
  4. 「OK」を選択してAPIキーを入力
  5. これで高精度AI認識が利用可能になります！
🎉 設定完了！
これで同じ物体は一貫して正確に認識されるようになります。認識精度が大幅に向上し、より実用的にお使いいただけます。

🔧 トラブルシューティング

よくある問題と解決方法

「API Error: 403」が表示される場合

  • APIキーが正しく入力されているか確認
  • Vision APIが有効化されているか確認
  • APIキーの制限設定を確認

「API Error: 429」が表示される場合

  • 月1000回の無料枠を超過した可能性
  • 課金設定を行うか、月が変わるまで待機

認識結果が期待と異なる場合

  • 写真の明るさや角度を調整
  • 物体をフレーム中央に配置
  • 背景をシンプルにする
APIキーをリセットしたい場合
  1. アプリの設定（ブラウザの開発者ツール）でローカルストレージをクリア
  2. または新しいAPIキーを作成して再設定
